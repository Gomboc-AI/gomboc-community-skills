# ORL supported `--language` IDs

These values are accepted by the ORL CLI (for example `walk`, `remediate`, and rule `metadata.language`). The authoritative list for your installed image is:

```bash
docker run gombocai/orl language
```

As of `gombocai/orl:latest` (`docker run gombocai/orl language`), supported languages are:

1. `bash`
2. `bicep`
3. `c`
4. `cloudformation-json`
5. `cloudformation-yaml`
6. `cpp`
7. `csharp`
8. `css`
9. `docker`
10. `elixir`
11. `go`
12. `gotemplate`
13. `groovy`
14. `hcl`
15. `helm`
16. `html`
17. `java`
18. `javascript`
