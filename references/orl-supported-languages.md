# ORL supported `--language` IDs

These values are accepted by the ORL CLI (for example `walk`, `remediate`, and rule `metadata.language`). The authoritative list for your installed image is:

```bash
docker run gombocai/orl language
```

As of the community skills baseline (matches `gombocai/orl language` output), supported languages are:

1. `bash`
2. `c`
3. `cloudformation-json`
4. `cloudformation-yaml`
5. `cpp`
6. `csharp`
7. `css`
8. `docker`
9. `elixir`
10. `go`
11. `gotemplate`
12. `groovy`
13. `hcl`
14. `helm`
15. `html`
16. `java`
17. `javascript`
18. `json`
